# ML/DL Beginner's Pack: A Composition of 3 beginner-friendly Models + Problems #

## Overview ##
This is my attempts on 3 well-known datasets: the IMDB movie dataset, the Reuters newswires dataset and the California housing dataset. The 3 problems are as follows:

```imdb.ipynb```: Binary Classification - determine whether a review is positive or negative (praise or criticism)

```newswires.ipynb```: Multiclass Classification - choosing the right categories for a newswire.

```houseprice.ipynb```: Regression - Predicting the price of a house. It's simple as that.

## Disclaimer ##

Some parts of the code in the 3 notebooks are taken from **Chapter 4** book **Deep Learning with Python** by *Francios Chollet* and *Matthew Watson*. However, I did make many changes in the codes by adding new models, taking different approaches as well as creating additional visualizations (graphs, charts, etc).

These notebooks are not created to be judged by their scores/ efficiencies. They are intended to provide a beginner approach to AI coding (not vibe coding) - how to actually write those Python syntax after learning about Linear Algebra, Matrix Multiplication, Probabilities and stuff like that. I struggle a lot with the Math I learn and how to actually code for a long time, and this is how I started.
